Introduction

In recent years, with the development of deep learning technology, infrared small target detection has made rapid progress in various military and civilian fields, such as marine resource utilization, high-precision navigation, autonomous driving, and ecological environment monitoring. However, with the increasing demands for edge-side applications such as micro-satellites, micro-UAVs, autonomous vehicles and ships, existing infrared small target detection networks struggle to achieve efficient response in edge-side scenarios with limited annotated samples and computational resources. These limitations hinder their practical value in commercial and engineering applications.
To address the aforementioned problems, we decide to organize a resource-limited infrared small target detection challenge. We aim to collaborate with peer institutions to further promote infrared small target detection in edge-side scenarios.


Description of the Challenge

The objective of this challenge is to detect small target in infrared images under limited resources: less annotation (i.e., Track 1: Weakly Supervised Infrared Small Target Detection Under Single Point Supervision) and smaller model size (i.e., Track 2: Fully Supervised Lightweight Infrared Small Target Detection Under Pixel-level Supervision).
During the model development phase, the training set will be released. Both original infrared images, the corresponding masks with single point annotation (track 1) and pixel-level annotation (i.e., track2) are available. The participants can train their models. During the test phase, the participants should submit their developed model and weights files. The organizers will evaluate the submitted models with a set of objective quantitative metrics.


Datasets, Evaluation Metrics and Baseline Model

Datasets

Seven public datasets SIRST-V2[1], IRSTD-1K [17], IRDST [9], NUDT-SIRST [7] NUDT-SIRST-Sea [12], NUDT-MIRSDT [8], Anti-UAV [4] and our self-developed dataset are used in this challenge. The dataset contains images with various target shapes (e.g., point target, spotted target, extended target), wavelengths (e.g., near-infrared, shortwave infrared and thermal), image resolution (e.g., 256, 512, 1024, 3200, etc.), at varied imaging systems (e.g., land-based, aerial-based and space-based imaging systems) Figure 1 shows some example images of the training sets. All datasets used in the challenges have been licensed by their authors, and the copyrights of all the datasets belong to their authors. Note that, the datasets provided are not allowed to be used beyond this challenge.

For track 1, 9300 images with groundtruth (GT) mask annotation are released for trainning and validation. 2000 images are used for test, and the groundtruth masks will not be released.
For track 2, 6000 images with coarse point annotation\( ^ { 1 } \) are released for training and validation. 500 images are used for test, and the groundtruth masks will not be released.
\( ^ { 1 } \)GT point is located around the centroid of the GT mask under Gaussian distribution

Evaluation Metrics

Pixel-level metric (i.e., intersection over union \( I o U\)) and target-level metrics (i.e., probability of detection \( P _ { d }\) and false-alarm rate \( F _ { a }\)) will be used as metrics for performance evaluation in tracks 1, 2, and their implementations can be found in our public toolbox BasicIRSTD\( ^ { 2 } \). Note that, the linear weighted summation of IoU and Pd is used for ranking, and the corresponding performance score Sp is defined as: $$ S _ { p } = \alpha \times I o U + ( 1 - \alpha ) \times P _ { d } , \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ ( 1 )$$ where \( \alpha \) represents the weight.
For track 3, besides the performance score, we also introduce the number of parameters \( P \) and \( FLOPs \) \( F \) for performance evaluation. Note that, we rank entries based on the sum of these three values, with each value normalized by a baseline state-of-the-art model for the task. The corresponding efficiency score \( S _ { e } \) is defined as: $$ S _ { e } = 1 - \frac { P _ { s u b } / P _ { b a s e } + F _ { s u b } / F _ { b a s e } } { 2 } , \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ ( 2 )$$ where \( \left[ \cdot \right] _{s u b}\) and \( \left[ \cdot \right] _{b a s e}\) represent the corresponding values of submission and baseline model. The final score \( S _ { p e }\) is the multiple weighted summations of the performance score and the efficiency score, and can be defined as: $$ S _ { p e } = \sum _ { n = 1 } ^ { N } \left[ \frac { n } { N } \times S _ { p } + ( 1 - \frac { n } { N } ) \times S _ { c } \right] / N . \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ ( 3 )$$ \( ^ { 2 } \)BasicIRSTD is a PyTorch-based open-source and easy-to-use toolbox for infrared small target detection (IRSTD). This toolbox introduces a simple pipeline to train/test your methods, and builds a benchmark to comprehensively evaluate the performance of existing methods.
